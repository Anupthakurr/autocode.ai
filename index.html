Setup Guide

How to get started

Get up and running in under 2 minutes. Just install, add your API key, and start building.

1

Install the Extension

Open VS Code, press Ctrl+P (or Cmd+P on Mac), and paste the install command:

VS Code — Quick Open (Ctrl+P)
ext install anupthakur.autocode-ai-anup

Or install directly from the VS Code Marketplace

2

Get Your Gemini API Key

Visit Google AI Studio to generate a free Gemini API key. It only takes a few seconds.

Google AI Studio aistudio.google.com/app/apikey
3

Add API Key to VS Code Settings

Open VS Code Settings (Ctrl+,), search for "AutoCodeAI", and paste your Gemini API key.

settings.json 
{
  "AutoCodeAI.apiKey": "your-gemini-api-key-here"
}
